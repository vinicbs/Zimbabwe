Objetivo:
-Meu trabalho tem como objetivo final mostrar quais são os países com maiores e menores expectativas de vida em diferentes epocas e fazer uma projeção para os próximos anos.

Observações:
-Neste programa, os objetos ainda estão sendo gerados de forma randomica para verificar os metodos de ordenação mais ageis.
-A classificação é feita por:
		-numericos: objetos são ordenados a partir da expectativa de vida, que fica em torno de 65 a 85 anos.
		-categoricos: objetos são ordenados a partir do nome do pais. Neste programa foram usados como exemplo os nomes dos paises da America do Sul.
-No trabalho final a lista de objetos tera em torno de 2000 elementos.
-Foi constatado que para o problema deste trabalho, o método de ordenação mais eficaz é o Shell Sort, seguido do Quick Sort Randomizado.

Implementar:
-Objetos serem gerados a partir do arquivo .xml contido neste repositorio.
-User interface.
